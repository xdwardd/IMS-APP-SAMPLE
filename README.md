<div>
  <h3>Installation</h3>
  <ul>
    <li>
      Clone the Repository:
      <p>git clone https://github.com/xdwardd/IMS-APP-SAMPLE.git</p>
    </li>
     <li>
       Navigate to the Project Directory:
      <p>cd ims-app-sample</p>
    </li>
     <li>
      Install Dependencies:
      <p>npm install</p>
    </li>
     <li>
     Start the Development Server:
      <p>npm run dev</p>
    </li>
  </ul>
